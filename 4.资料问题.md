# 证据收集完整清单

> 离职后账号会被注销或踢出，很多记录将永久消失。现在就要行动，别拖。

---

## 一、证明劳动关系存在

| 材料 | 获取方式 | 重要程度 |
|------|---------|---------|
| 劳动合同 | 自己保存的纸质版或电子版 | ★★★★★ |
| 入职通知书/offer | 邮件或钉钉消息截图 | ★★★★ |
| 工牌、员工证 | 实物拍照 | ★★★ |
| 社保缴纳记录 | 支付宝查询或社保局打印 | ★★★★★ |
| 公积金缴纳记录 | 当地公积金APP或官网 | ★★★★ |
| 工资条 | 每月截图或拍照保存 | ★★★★★ |
| 银行工资流水 | 去银行柜台打印近1-2年 | ★★★★★ |

---

## 二、证明工作地在实际工作城市

| 材料 | 获取方式 |
|------|---------|
| 钉钉/企业微信打卡记录 | 截图（带定位和日期），**今天就截** |
| 工作群带定位的消息 | 截图保存 |
| 工牌或门禁卡上的地址 | 拍照 |
| 劳动合同上注明的工作地址 | 已有 |
| 当地租房合同/居住证 | 已有 |
| 报销记录（当地交通、餐饮） | 钉钉审批截图 |

---

## 三、证明工作内容和工龄

| 材料 | 说明 |
|------|------|
| 历史工资条 | 证明连续在职时间，是计算N的依据 |
| 社保连续缴纳记录 | 起始时间很关键，决定工龄 |
| 工作邮件、钉钉消息 | 证明实际在岗 |
| Git提交记录 | 时间戳客观，可导出 |
| 项目文档、工作成果截图 | 证明工作内容 |
| 绩效考核记录 | HR系统截图 |

---

## 四、证明被裁这件事

| 材料 | 说明 |
|------|------|
| 公司书面通知 | 邮件、钉钉消息、纸质文件，截图或拍照 |
| 谈话录音 | 手机放口袋，全程录 |
| 对方提出的赔偿方案 | 截图或拍照，**不要签字** |
| 协商解除协议 | 拍照留存，先不签 |

---

## 五、证明加班（间接证据）

由于考勤数据员工无权查看，用以下间接证据：

| 材料 | 说明 |
|------|------|
| 非工作时间的工作消息 | 晚上10点、周末发的工作消息截图 |
| 领导非工作时间布置任务的记录 | 截图，时间戳是关键 |
| Git非工作时间的提交记录 | 导出commit log |
| 外卖订单记录 | 美团/饿了么，加班时在公司附近点的 |
| 打车记录 | 滴滴/高德，深夜从公司附近出发的订单 |

> 考勤数据由公司掌握，仲裁时可申请仲裁委向公司调取，公司拒绝提供则承担不利后果。

---

## 六、同事证明材料

- 请1-2个同城同事写情况说明并签字（证明共同在当地工作）
- 保留和同事的工作沟通记录

---

## 七、离职后会消失的内容（立即截图）

以下内容离职后将无法访问，**今天就要截图保存到本地**：

**钉钉（最危险）：**
- 打卡记录和定位历史
- 加班申请记录
- 工作群聊天记录
- 审批记录（年假、报销等）
- 工资条（如果公司通过钉钉发的）

**企业微信：**
- 和同事、领导的聊天记录
- 工作群消息
- 打卡记录

**公司内网系统：**
- HR系统里的年假余额、出勤记录
- 绩效考核页面
- 报销记录

**公司邮箱：**
- 工作邮件转发到私人邮箱（只转劳动关系相关的，不要带保密文件）

---

## 优先级排序

社保记录 > 劳动合同 > 工资流水 > 打卡记录 > 谈话录音

社保和工资流水是最客观的，其他都是辅助。
